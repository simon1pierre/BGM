# Settings Center

The settings center lives at `/beacons/admin/settings` and controls:

## Site Identity
- Site name, tagline, description
- Logo and favicon

## Contact & Social
- Email, phone, address
- YouTube, Facebook, Instagram

## Homepage Controls
- Hero title/subtitle + CTA labels/links
- Featured/recommended limits

## Content Defaults
- Default per-page
- Auto publish

## Moderation
- Auto-approve comments
- Require name/email
- Block words

## Analytics
- Enable tracking
- Retention days
- Anonymize IP

## Maintenance & Security
- Maintenance mode + message
- Force admin 2FA
- Session timeout
